# Vite PROD-билд для сайта

Этот проект — обёртка вокруг твоего текущего сайта, собранная под **Vite 5**:
- мультистраничная сборка (все `*.html` в проекте собираются как отдельные входы),
- автоматическая минификация и хеширование ассетов,
- поддержка `public/` (картинки/шрифты копируются как есть),
- `src/` — исходники CSS/JS (переведены на `type="module"`).

## Быстрый старт

```bash
npm i
npm run dev
```
Открой указанную ссылку из консоли (обычно http://localhost:5173).

## Продакшн-сборка

```bash
npm run build
npm run preview
```

Собранные файлы появятся в `dist/`. Деплой: загрузи содержимое `dist/` на сервер.

## Структура

- `src/` — исходники CSS/JS. Я переподключил их в HTML через `/src/...` и `type="module"` (Vite сам их соберёт, минифицирует и вставит правильные хэши).
- `public/` — изображения и статические файлы, которые не нужно прогонять через сборку.
- `*.html` — страницы сайта (поддерживается многостраничник).
- `vite.config.ts` — конфиг Vite: автопоиск всех HTML и сборка мультистраничника.
- `postcss.config.js` — Autoprefixer.
- `package.json` — скрипты `dev/build/preview`.

## Замечания

- В HTML я обновил ссылки на локальные `.css/.js` → `/src/...` (для внешних CDN-файлов ссылки оставлены как есть).
- `type="module"` добавлен к `<script>` с локальными путями, чтобы Vite корректно их обработал.
- Все прочие файлы перенесены в `public/`, пути в HTML не менялись (Vite сам их скопирует).
